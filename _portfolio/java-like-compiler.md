Course project for Compiler Design and Implementation.

- Designed and implemented a compiler for a Java-like teaching language.
- Covered syntax and semantic checking, code generation, and IR optimization.
- Implemented passes including mem2reg, register allocation, function inlining, dead code elimination, and global variable localization.
